API Engineering (FastAPI) — Project Recipe Card

Per-endpoint 5-step · layered arch · auth · DB · validation · middleware · streaming · OWASP. Deep companion: api_engineering_cheatsheet.md · Stack: FastAPI ≥0.115 · Pydantic v2 · SQLAlchemy 2.0 · Python 3.12+

◆ Per-endpoint 5-step pattern

  1. DECLARE — verb, path, response_model, status, OpenAPI tags
  2. VALIDATE — Pydantic request body @ boundary (422 on bad data)
  3. INJECT — Annotated[T, Depends(...)]: session, settings, user
  4. DELEGATE — service does the actual work (routes don't touch SQL)
  5. RETURN — value conforming to response_model (filters output)
Smell test: if the route imports select from SQLAlchemy, SQL has leaked out of the repository layer.

◆ Canonical endpoint (paste-able)

@router.post("/trades", response_model=TradeRead, status_code=status.HTTP_201_CREATED) async def create_trade( payload: TradeCreate, # 2. validate session: SessionDep, # 3. inject (DB) user: CurrentUser, # inject (auth+identity) ) -> TradeRead: return await TradeService(session).create(payload, owner_id=user.id) # 4 + 5

◆ Project-level wiring (once at startup)

  1. settings — pydantic-settings + @lru_cache; read env + .env in dev
  2. lifespan — create_async_engine + sessionmaker on startup; engine.dispose() on shutdown
  3. middleware — CORS, TrustedHost, GZip, security headers (order = onion; LAST added = FIRST hit)
  4. routers — include versioned modules (app/api/v1/…) with prefix + tags
  5. handlers — register NotFoundError→404, ConflictError→409, etc.

◆ Request flow (mental model)

REQUEST → middleware (CORS · rate-limit · security headers · request-id) → route (Depends graph resolved · Pydantic validates payload) → service (business logic · authz · calls repo) → repository (SQL via select() + execute · returns ORM rows) → service (compose, transform) → route returns → response_model filters → response_class serialises RESPONSE
Raise propagates UP until caught by an exception handler (§10) or becomes a 500. Services raise domain exceptions (NotFoundError, ConflictError) — NEVER HTTPException. HTTP mapping is the routes' job.

● Lifespan (startup / shutdown)

@asynccontextmanager async def lifespan(app: FastAPI): # STARTUP engine = create_async_engine(settings.database_url, pool_pre_ping=True, pool_size=10, max_overflow=20) app.state.session_factory = async_sessionmaker(engine, expire_on_commit=False) # warm caches, ping deps, etc. yield # SHUTDOWN await engine.dispose() app = FastAPI(lifespan=lifespan)

● ORM models (SQLAlchemy 2.0)

from sqlalchemy.orm import DeclarativeBase, Mapped, mapped_column, relationship from sqlalchemy import String, ForeignKey, DateTime, Numeric from datetime import datetime, timezone from decimal import Decimal class Base(DeclarativeBase): pass class User(Base): __tablename__ = "users" id: Mapped[int] = mapped_column(primary_key=True) email: Mapped[str] = mapped_column(String(254), unique=True, index=True) password_hash: Mapped[str] = mapped_column(String(256)) created_at: Mapped[datetime] = mapped_column( DateTime(timezone=True), default=lambda: datetime.now(timezone.utc)) # datetime.utcnow deprecated in 3.12+ trades: Mapped[list["Trade"]] = relationship( back_populates="owner", cascade="all, delete-orphan") class Trade(Base): __tablename__ = "trades" id: Mapped[int] = mapped_column(primary_key=True) owner_id: Mapped[int] = mapped_column(ForeignKey("users.id"), index=True) symbol: Mapped[str] = mapped_column(String(10), index=True) qty: Mapped[Decimal] = mapped_column(Numeric(18, 4)) price: Mapped[Decimal] = mapped_column(Numeric(18, 4)) owner: Mapped["User"] = relationship(back_populates="trades")
Use Mapped[T] + mapped_column(...) (2.0 style). Legacy Column() still works but loses typing.

● Async DB sessions + 2.0 queries

Engine + session factory

Query patterns
from sqlalchemy import select, update, delete, func # SCALAR — single column email = await session.scalar(select(User.email).where(User.id == uid)) # SCALARS — list of ORM objects trades = (await session.scalars( select(Trade).where(Trade.owner_id == uid).order_by(Trade.created_at.desc()) )).all() # COUNT n = await session.scalar(select(func.count()).select_from(Trade)) # JOIN stmt = select(Trade, User).join(User, Trade.owner_id == User.id) rows = (await session.execute(stmt)).all() # UPDATE / DELETE await session.execute(update(Trade).where(Trade.id == tid).values(price=p)) await session.commit()
Deprecated: session.query(Model).filter(...).all(). Always select() + execute/scalars/scalar.
Eager loading (avoid N+1)
from sqlalchemy.orm import selectinload, joinedload stmt = select(User).options(selectinload(User.trades)) # SELECT IN (separate query) stmt = select(User).options(joinedload(User.trades)) # LEFT OUTER JOIN

◆ Auth — JWT + Argon2 (canonical 2026)

Password hashing (pwdlib + Argon2id)
from pwdlib import PasswordHash from pwdlib.hashers.argon2 import Argon2Hasher password_hash = PasswordHash((Argon2Hasher(),)) def hash_pw(plain: str) -> str: return password_hash.hash(plain) def verify_pw(plain: str, hashed: str) -> bool: return password_hash.verify(plain, hashed)
Argon2id (memory-hard) is OWASP's current first choice. Bcrypt OK as a fallback.
JWT issuance + decode (PyJWT — canonical 2026)
import jwt # uv add pyjwt[crypto] from datetime import datetime, timedelta, timezone import uuid ALG = "HS256" ISS = "api.example.com" AUD = "api.example.com" def create_token(sub: str, secret: str, minutes: int = 15) -> str: now = datetime.now(timezone.utc) payload = { "sub": sub, "iat": now, "exp": now + timedelta(minutes=minutes), "iss": ISS, "aud": AUD, # prevent cross-service replay "jti": str(uuid.uuid4()), # token id (for revocation list) } return jwt.encode(payload, secret, algorithm=ALG) def decode_token(tok: str, secret: str) -> dict: try: return jwt.decode(tok, secret, algorithms=[ALG], # explicit list defends alg=none audience=AUD, issuer=ISS, options={"require": ["exp", "iat", "iss", "aud", "sub"]}) except jwt.PyJWTError: raise CredentialsError
Don't use python-jose — unmaintained since 2024 with unpatched CVEs (CVE-2024-33663 algorithm-confusion, CVE-2024-33664 DoS). Use PyJWT (or authlib).
Refresh-token rotation (prod auth)
# Short-lived ACCESS (15m) + long-lived REFRESH (7d) with rotation + reuse detection. @router.post("/token/refresh") async def refresh(refresh_token: str, session: SessionDep, settings: SettingsDep): payload = decode_token(refresh_token, settings.secret_key.get_secret_value()) jti = payload["jti"] if await is_revoked(jti): # reuse detection await revoke_user_all(payload["sub"]) # SECURITY: rotate everything raise CredentialsError await revoke(jti) # one-time use → blacklist old jti return { "access_token": create_token(payload["sub"], ..., minutes=15), "refresh_token": create_token(payload["sub"], ..., minutes=60*24*7), "token_type": "bearer", }
OAuth2 dep (`get_current_user`)
from fastapi.security import OAuth2PasswordBearer, OAuth2PasswordRequestForm oauth2_scheme = OAuth2PasswordBearer(tokenUrl="/v1/auth/token") async def get_current_user( token: Annotated[str, Depends(oauth2_scheme)], session: SessionDep, settings: SettingsDep, ) -> User: payload = decode_token(token, settings.secret_key.get_secret_value()) user = await session.scalar(select(User).where(User.id == int(payload["sub"]))) if user is None: raise CredentialsError return user CurrentUser = Annotated[User, Depends(get_current_user)]
Login route (OAuth2 form)
@router.post("/token") async def login( form: Annotated[OAuth2PasswordRequestForm, Depends()], session: SessionDep, settings: SettingsDep, ): user = await session.scalar(select(User).where(User.email == form.username)) if user is None or not verify_pw(form.password, user.password_hash): raise HTTPException(status_code=401, detail="Bad credentials", headers={"WWW-Authenticate": "Bearer"}) secret = settings.secret_key.get_secret_value() return {"access_token": create_token(str(user.id), secret, minutes=15), "refresh_token": create_token(str(user.id), secret, minutes=60*24*7), "token_type": "bearer"}

◆ Authorization — close BOLA (OWASP API1)

# Every read/write of an object MUST verify owner_id (or role) in the SERVICE. class TradeService: def __init__(self, session: AsyncSession): self.s = session async def get(self, trade_id: int, user_id: int) -> Trade: t = await self.s.scalar(select(Trade).where(Trade.id == trade_id)) if t is None: raise NotFoundError("Trade") if t.owner_id != user_id: # ← BOLA check raise ForbiddenError("Not your trade") return t
BOLA = Broken Object-Level Authz = #1 API risk (OWASP). Default-deny. Every GET / PATCH / DELETE that takes an ID must verify ownership before returning. ID-only checks (request path parses to int) are NOT enough.

◆ Error handling — one mapping

# app/exceptions.py — domain exceptions class NotFoundError(Exception): pass class ConflictError(Exception): pass class ForbiddenError(Exception): pass class CredentialsError(Exception): pass # app/main.py — single registry, every domain exception → HTTP code from fastapi import Request from fastapi.responses import JSONResponse @app.exception_handler(NotFoundError) async def _nf(req: Request, exc: NotFoundError): return JSONResponse(status_code=404, content={"detail": str(exc) or "Not found"}) @app.exception_handler(ConflictError) async def _conf(req: Request, exc: ConflictError): return JSONResponse(status_code=409, content={"detail": str(exc) or "Conflict"}) @app.exception_handler(ForbiddenError) async def _fb(req: Request, exc: ForbiddenError): return JSONResponse(status_code=403, content={"detail": "Forbidden"}) # Pydantic ValidationError handled automatically by FastAPI → 422
NEVER include stack traces / SQL / internal error text in JSON. Log it (with request_id), return a generic message.

● Config — pydantic-settings + @lru_cache

from pydantic import SecretStr, PostgresDsn from pydantic_settings import BaseSettings, SettingsConfigDict from functools import lru_cache class Settings(BaseSettings): model_config = SettingsConfigDict(env_file=".env", extra="ignore", case_sensitive=False) # nested: ENV var "DB__POOL_SIZE" overrides db.pool_size database_url: PostgresDsn secret_key: SecretStr # hidden in repr/logs cors_origins: list[str] = [] rate_limit_per_minute: int = 100 @lru_cache def get_settings() -> Settings: return Settings()

● Middleware stack (order = onion)

# LAST added = FIRST to hit the request (and LAST to see the response). # Add OUTER → INNER: app.add_middleware(GZipMiddleware, minimum_size=500) app.add_middleware(TrustedHostMiddleware, allowed_hosts=["api.example.com"]) app.add_middleware(CORSMiddleware, allow_origins=settings.cors_origins, # specific URLs only allow_credentials=True, allow_methods=["GET","POST","PATCH","DELETE"], allow_headers=["Authorization","Content-Type"]) # Custom security headers + request-id middleware: @app.middleware("http") async def sec_headers(request: Request, call_next): rid = request.headers.get("X-Request-Id") or str(uuid.uuid4()) request.state.request_id = rid resp = await call_next(request) resp.headers["X-Request-Id"] = rid resp.headers["X-Content-Type-Options"] = "nosniff" resp.headers["X-Frame-Options"] = "DENY" resp.headers["Referrer-Policy"] = "strict-origin-when-cross-origin" resp.headers["Strict-Transport-Security"] = "max-age=63072000; includeSubDomains" return resp
CORS: NEVER allow_origins=["*"] with allow_credentials=True — silently dropped by browsers. Use explicit origins.

● Pagination + rate limiting (OWASP API4)

Cursor pagination (canonical for large tables)
@router.get("/trades", response_model=Page[TradeRead]) async def list_trades( session: SessionDep, user: CurrentUser, limit: Annotated[int, Query(ge=1, le=100)] = 20, cursor: Annotated[int | None, Query()] = None, ): stmt = (select(Trade).where(Trade.owner_id == user.id) .order_by(Trade.id.desc()).limit(limit + 1)) if cursor: stmt = stmt.where(Trade.id < cursor) rows = (await session.scalars(stmt)).all() next_cursor = rows[limit].id if len(rows) > limit else None return Page(items=rows[:limit], next_cursor=next_cursor)
Cursor is stable under inserts (offset isn't). Always cap limit with le=.

NDJSON streaming (large CSV / generators)

from sse_starlette import EventSourceResponse # uv add sse-starlette @router.get("/stream") async def stream(user: CurrentUser): async def gen(): async for chunk in llm.astream("..."): yield {"event": "message", "data": chunk.content} return EventSourceResponse(gen())

WebSockets (bidirectional)
from fastapi import WebSocket, WebSocketDisconnect @app.websocket("/ws") async def ws(websocket: WebSocket): await websocket.accept() try: while True: msg = await websocket.receive_json() await websocket.send_json({"echo": msg}) except WebSocketDisconnect: pass

◆ Observability

import structlog, time, uuid log = structlog.get_logger() @app.middleware("http") async def access_log(request: Request, call_next): t0 = time.perf_counter() rid = getattr(request.state, "request_id", str(uuid.uuid4())) try: resp = await call_next(request) log.info("http", request_id=rid, method=request.method, path=request.url.path, status=resp.status_code, latency_ms=round((time.perf_counter()-t0)*1000, 2)) return resp except Exception: log.exception("http_error", request_id=rid) raise

◆ Deploy — uvicorn + gunicorn

# Production process: gunicorn manages N uvicorn workers uv run gunicorn app.main:app \ --workers 4 \ # N = 2 × CPU + 1 (sync) or CPU (async) --worker-class uvicorn.workers.UvicornWorker \ --bind 0.0.0.0:8000 \ --timeout 30 \ --graceful-timeout 30 \ --keep-alive 5 \ --access-logfile - --error-logfile - # Docker (slim multi-stage): FROM python:3.12-slim AS base RUN pip install uv WORKDIR /app COPY pyproject.toml uv.lock ./ RUN uv sync --frozen --no-dev COPY . . EXPOSE 8000 CMD ["uv","run","gunicorn","app.main:app", "--workers","4","--worker-class","uvicorn.workers.UvicornWorker", "--bind","0.0.0.0:8000"]
  • Run behind a reverse proxy (nginx / Traefik) for TLS termination + buffering.
  • HSTS + HTTPSRedirectMiddleware at the proxy side.
  • Two healthchecks: /healthz (liveness, cheap — process alive) vs /readyz (readiness — does DB ping). Conflating them causes restart storms on transient DB blips.
  • Workers: 2×CPU+1 for sync; for ASYNC use CPU count (over-subscribing wastes connections).
  • Migrations: separate step (NOT at lifespan), run once per deploy before workers start.

▲ Top anti-patterns (6-month memory)

  • SQL in route → bypass service; can't unit-test business logic.
  • HTTPException in service → couples business code to HTTP. Use domain exceptions.
  • response_model missing → output leaks password_hash, internal IDs.
  • Sync DB driver (psycopg2) in async path → entire worker blocks. Use asyncpg.
  • on_event("startup") deprecated → use lifespan.
  • x = Depends(...) deprecated → use Annotated[T, Depends(...)].
  • session.query() legacy → use select() + execute/scalars.
  • Column() legacy → use Mapped[] + mapped_column.
  • Plain str for password/email/URL → use SecretStr / EmailStr / HttpUrl.
  • Stack traces in JSON errors → leaks internals. Log it, return generic.
  • BackgroundTasks for "important" jobs → lost on worker crash. Celery / RQ instead.
  • CORSMiddleware allow_origins=["*"] with credentials → silently dropped.
  • Forgetting BOLA check → user A reads user B's data. Default-deny in service.
  • Forgetting .commit() in async session → writes silently lost. expire_on_commit=False in factory.
  • N+1 query → use selectinload / joinedload on relationships.
  • python-jose unmaintained + CVEs → use PyJWT.
  • datetime.utcnow deprecated 3.12+ → datetime.now(timezone.utc).
  • slowapi without request: Request in route signature → silent no-op. Required as first non-self param.
  • asyncpg + PgBouncer transaction pooling conflicts (prepared statements). Use session pooling, or switch to psycopg[binary,pool] v3.
  • cors_origins=[] env parsing: pydantic-settings expects JSON (CORS_ORIGINS='["a","b"]'), not comma-sep. Add a @field_validator if you want commas.
  • One /health doing DB ping → restart storms on transient blips. Split /healthz (liveness) vs /readyz (readiness).
