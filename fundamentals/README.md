# Fundamentals

Standalone STEM notebooks — not strictly part of the ML / quant track, but
useful refresher material when the maths or science behind a model is
shaky.

## Contents

- `mathematics.ipynb` — linear algebra, calculus, probability essentials.
- `physics.ipynb` — physics intuitions that map onto ML (energy / gradient / equilibrium).
- `chemistry.ipynb` — molecular property notebooks (useful intro for cheminformatics).
- `visual_exercises/` — visual / geometric intuition builders.

## How to use

These are **reference** — open them when you hit a concept you can't picture,
not as a sequential course. Each notebook is standalone.

## Why this module exists

When the maths in a quant or ML paper isn't sinking in, drop into the
relevant notebook here for the intuition before re-attempting the paper.
Companion to the
[quant notebook teaching style](../quant_finance/) — narrative companion
notebooks (`00_*_intuition.ipynb`) live in each quant subject dir; this is
the wider STEM equivalent.
