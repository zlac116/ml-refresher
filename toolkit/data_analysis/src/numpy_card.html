NumPy + SciPy — Reference Card

arrays · indexing · broadcasting · axis · linalg · RNG · ufuncs + scipy stats/optimize/sparse. Stack: numpy ≥2.0 · scipy ≥1.14 · Python ≥3.10 · Modern: np.random.default_rng(), explicit axis

◆ Master pattern (every numpy script)

  1. RNG — rng = np.random.default_rng(seed) (not np.random.seed)
  2. Create — np.zeros / arange / linspace / from list. Declare dtype.
  3. Reshape — .reshape(-1, n) for -1 auto-dim
  4. Index / mask — basic slice, boolean, fancy; know view-vs-copy
  5. Vectorise — ufuncs; specify axis= on reductions
  6. Linalg — solve not inv @; specific factorisations
Mental model: think in ARRAYS + AXES, not loops. axis=0 collapses rows (result has cols); axis=1 collapses cols (result has rows).

▲ Broadcasting (the key insight)

# Rule: align shapes from the RIGHT. Pad missing left dims with 1. # Compatible if each dim is EQUAL or ONE of them is 1. a = np.zeros((3, 5)) # (3, 5) b = np.arange(5) # (5,) → (1, 5) → broadcasts to (3, 5) a + b # OK col = np.arange(3).reshape(3, 1) # (3, 1) row = np.arange(5) # (5,) → (1, 5) col + row # (3, 5) — outer sum # Add a trailing dim via None / np.newaxis for explicit intent X = np.zeros((100, 5)) mean = X.mean(axis=0, keepdims=True) # (1, 5) — safer than mean.shape (5,) X - mean # broadcasts (100, 5) − (1, 5) = (100, 5)
Keep keepdims=True on reductions when you plan to broadcast back — avoids a lot of .reshape.

▲ Linear algebra

A @ B # matmul (prefer @ over np.dot for 2D) np.dot(a, b) # 1D dot; 2D matmul (equiv to @) np.linalg.norm(x, ord=2, axis=0) # L2 by column # Solve Ax = b — NEVER compute inv(A) @ b explicitly x = np.linalg.solve(A, b) # canonical x = np.linalg.lstsq(A, b, rcond=None)[0] # least-squares (over/underdetermined) # Decompositions U, s, Vt = np.linalg.svd(A, full_matrices=False) # SVD (economy form) L = np.linalg.cholesky(A) # A = L Lᵀ (PSD only) Q, R = np.linalg.qr(A) # QR w, V = np.linalg.eig(A) # eigenvalues + right eigenvectors w, V = np.linalg.eigh(A) # SYMMETRIC A — faster + stable # Matrix inverse — rarely needed; use solve instead np.linalg.inv(A) # explicit inverse np.linalg.pinv(A) # Moore-Penrose pseudo-inverse # Determinant / trace np.linalg.det(A); np.trace(A) # Batched linalg (broadcasting on last two dims) np.linalg.solve(A_stack, b_stack) # A_stack.shape = (B, n, n) → x.shape (B, n) # einsum — the daily-driver for batched / non-obvious linalg np.einsum("ij,jk->ik", A, B) # ≡ A @ B np.einsum("ii->i", A) # diagonal np.einsum("ii->", A) # trace np.einsum("i,j->ij", a, b) # outer product np.einsum("bij,bjk->bik", A_stack, B_stack) # BATCHED matmul (B pairs) np.einsum("bi,bi->b", x, y) # per-row inner product (dot batch)
Never inv(A) @ b — numerically unstable + slower than solve(A, b). Use solve; for many RHS with same A use scipy.linalg.lu_factor + lu_solve.

▲ Top traps (6-month memory)

  • np.random.seed() legacy global; use rng = np.random.default_rng(seed).
  • Missing axis= — x.sum() collapses ALL axes; almost always want axis=0/1.
  • View vs copy — basic slice = view; fancy/boolean = copy. Check b.base is a.
  • inv(A) @ b numerically bad. Use np.linalg.solve(A, b).
  • Float precision: a == b unsafe. np.isclose(a, b, rtol=1e-5).
  • NaN propagation: x.mean() → NaN if any NaN. Use np.nanmean.
  • x.std(ddof=1) for SAMPLE std (default ddof=0 = population).
  • np.empty = uninitialised garbage; fill before read.
  • Broadcasting aligns FROM THE RIGHT. Use keepdims=True.
  • Concat in a loop = O(N²). Build list, one final np.concatenate.
  • Sparse: never .toarray() a huge matrix. Stay in sparse space.
  • NumPy 2.0 dtype: Python floats → float64; ints → intp. Cast explicitly for cross-platform.
  • curve_fit without p0: silent bad local min. Always give a starting guess.
