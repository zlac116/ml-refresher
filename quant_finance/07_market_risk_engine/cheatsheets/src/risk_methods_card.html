◆ Master Recipe (4 steps)

  1. Identify factors — rate, spread, vol, FX, equity (bucket KRD per tenor)
  2. Generate shocks — history / parametric (z·σ) / MC (Cholesky)
  3. Revalue — sensitivity (linear) OR full-reval (exact)
  4. Aggregate — quantile (VaR) / mean-of-tail (ES) / gradient (MVaR)

◆ Three patterns

P1 — Sensitivity-based

risk ≈ s·shock or √(sᵀΣs)
Fast O(#factors); falls down on non-linearity.

P2 — Full revaluation

risk = T( V(shocked) − V(base) over many shocks )
Exact; cost = n_paths × pricer cost.

P3 — Empirical distribution

risk = T( empirical P&L vector )
No distribution assumption; bounded by window.
Historical = P2+P3; Parametric = P1; MC = P2+simulated; FHS = P2+P3+GARCH.

● I — VaR & Expected Shortfall

Historical VaR (§1)
VaR(α) = −sorted(P&L)[⌊(1−α)·N⌋]
P3 — empirical. No distribution assumption.
np.percentile(pnl, (1-conf)*100) * -1
Limits: bounded by window; vol-blind; calm year → tiny VaR.
Parametric / delta-normal VaR (§2)
σ² = sᵀ Σ s VaR(α) = z(α) · σ
P1 — analytical Normal. Fast.
norm.ppf(conf) * np.sqrt(s @ cov @ s)
Assumes: factors Normal + book LINEAR. Misses fat tails + gamma.
Monte Carlo VaR (§3)
L = chol(Σ); shocks = z·Lᵀ ~ N(0,Σ) P&L_k = V(base + shock_k) − V₀ (FULL reval) VaR(α) = −percentile(P&L, (1−α)·100)
P2 + simulated. Captures gamma + custom distributions.
L = np.linalg.cholesky(cov) z = rng.standard_normal((n,4)) shocks = z @ L.T
Always set seed for reproducible risk numbers.
Expected Shortfall (CVaR) (§4)
Empirical: ES_α = −mean(sorted[:⌈(1−α)N⌉]) Normal: ES_α = σ · φ(z(α)) / (1−α) FRTB: ES at α=0.975
Coherent (subadditive). Mean of tail, not boundary.
k = int(np.ceil((1-conf)*len(pnl))) -np.sort(pnl)[:k].mean()
ES/VaR (Normal): α=0.99 → 1.145×; α=0.975 → 1.193×.
Filtered Historical Sim (FHS) (§5)
shock_today = (σ_today / σ_then) · F_i then proceed as historical VaR
P2+P3+GARCH rescale. Solves "stale during crises".
Pitfall: GARCH mis-spec → wrong rescale. Plot σ̂_t.
Cornish-Fisher VaR (§6)
z_CF = z + (z²−1)S/6 + (z³−3z)K/24 − (2z³−5z)S²/36 VaR = −(μ + z_CF·σ)
Quick fat-tail fix without MC. S=skew, K=excess kurt.
Falls apart for |S|>2 or K>10 — use EVT/t-copula.
Marginal / Component / Incremental VaR (§7)
MVaR_i = z(α)·(Σw)_i / √(wᵀΣw) CVaR_i = w_i · MVaR_i [Σ CVaR_i = VaR ✓ Euler] IVaR_i = VaR(w) − VaR(w_{−i})
M = per-$ trade sizing; C = limit attribution; I = "if I close X".
Σ here = POSITION-return cov (N×N), NOT factor cov. Σ_pos = B·Σ_factor·Bᵀ.
Risk Budgeting / ERC (§7b)
solve w : w_i·(Σw)_i / (wᵀΣw) = b_i ∀i ERC special case: b_i = 1/N Maillard iteration: w_i ← target/(w_i·(Σw)_i); renormalise
Inverse of §7. SAA when expected returns uncertain. Risk-parity funds (Bridgewater).
w_new = w * (target / (w * (cov @ w))) w_new /= w_new.sum()
Stressed VaR (sVaR) (§8)
Historical VaR on FIXED 12M stress window window chosen for MAX sVaR on CURRENT book Basel II.5: Capital = max(VaR_t, k·avg₆₀) + max(sVaR_t, k·avg₆₀_sVaR)
Example: calm 250d → VaR 2.3; 2008 window → sVaR 9.2 → capital ~5× higher.

▲ II — Backtesting

Kupiec POF (§9)
LR = 2[x·ln(p/π) + (T−x)·ln((1−p)/(1−π))] ~ χ²(1); reject if LR > 3.841 π = 1−α, p = x/T, x = #hits
Unconditional hit-rate test. Ignores clustering.
Christoffersen independence (§10)
LR_ind ~ χ²(1) on 2×2 transition table LR_CC = LR_POF + LR_ind ~ χ²(2)
Catches vol-blind model: 6 hits OK count but all in 1 week.

ES backtest (Acerbi-Szekely T2) (§12)
Z = (1/T)·Σ[X·I/(ES·(1−α))] − 1 H₀: E[Z]=0; Z < 0 (negative) = under-est
Mandated by FRTB IMA. Low power at α=0.975 (~6 hits / 250d).

■ III — Risk Sensitivities

First-order (DV01, CS01, vega, FX-Δ, IE01) (§13)
s_f = [V(base + δ_f) − V(base)] / δ_f ΔV ≈ Σ_f s_f · shock_f
GreekFactorBump
DV01parallel rate+1 bp
CS01credit spread+1 bp
IE01breakeven inflation+1 bp
FX-ΔFX shock+1% (0.01)
Vegaimplied vol+1 vol pt (0.01)
Convention: "DV01 = V_bumped − V_base" → long bond DV01 is NEGATIVE. The other convention (loss-given-up) flips sign. Pick one and LABEL it.
Second-order (gamma, convexity, cross-gamma) (§14)
g_f = [V(+δ) − 2V₀ + V(−δ)] / δ² own g_{ab} = [V(+a+b) − V(+a−b) − V(−a+b) + V(−a−b)] / (4·δ_a·δ_b) cross
Drill 12 gammas: (1.86e-4, 4.29e-5, −1.06, 412.92). Adding 2nd-order shrank residual 4×.
Key Rate Duration (KRD) (§15)
KRD_k = V(bump z(T_k) +1bp) − V(base) Σ KRD_k ≈ parallel DV01
Curve-shape risk. Hedge vector, not scalar.
Pension fail mode: 5y IRS hedging 30y liab = parallel-neutral, drowns in steepener.

◆ IV — P&L Attribution

Greek-based Taylor explain (§18)
first_f = s_f · Δf second_f = ½ · g_f · Δf² unexp = full − Σfirst − Σsecond
Drill 12 result: full −3.024 / first −3.054 / 2nd +0.023 / unexp +0.008.
Rule of thumb: |unexp| < 5% |full| for vanilla derivs.
Risk-factor explain (RFE) (§19)
Sens RFE: contrib_f = s_f · Δf (order-indep, misses cross) Full RFE: contrib_f = V(only Δf) − V(base) (+ residual) Sequential: apply IN ORDER; Σ = full, order-dep
Vanilla → sens. Options → full + cross-gamma report. Regulator → sequential.
Actual / Hyp / RT P&L (§20)
Actual = V_eod(pos_eod, mkt_eod) − V_sod(pos_sod, mkt_sod) Hyp = V(pos_sod, mkt_eod) − V(pos_sod, mkt_sod) RT (FRTB) = V_VaR_pricer(pos_sod, …) Unexp_intraday = Actual − Hyp PLA-residual = Hyp − RT
Two residuals: desk behaviour (unexp_intraday) vs model quality (PLA-residual).
FRTB PLA test (§21)
ρ_s ≥ 0.80 / 0.70 / <0.70 = G/A/R KS < 0.09 / 0.12 / ≥0.12 = G/A/R Joint zone = WORST of the two
RED → desk excluded from IMA → SA capital fallback.
spearmanr(hyp, rt); ks_2samp(hyp, rt)
Carry / pull-to-par / theta (§22)
Time_PnL = V(mkt_today, t+1) − V(mkt_today, t) = carry + pull_to_par + theta + cross-roll
"What would have happened with NO market move." Carry is EXPECTED daily P&L.

● V — Estimation Inputs

Historical vol (§23)
σ̂² = (1/(N−1)) Σ(r − r̄)² σ̂_annual = σ̂_daily · √252
Equal weights. Stale during regime shifts.
EWMA (RiskMetrics) (§24)
σ̂²_t = λ·σ̂²_{t-1} + (1−λ)·r²_{t-1} λ = 0.94 daily, 0.97 monthly half-life = ln(0.5)/ln(λ) = 11.2d @ 0.94
One parameter; vol updates immediately. No mean reversion.
GARCH(1,1) (§25)
σ̂²_t = ω + α·r²_{t-1} + β·σ̂²_{t-1} σ²_∞ = ω/(1−α−β), α+β < 1 stationary α+β = 1 ≡ EWMA (IGARCH)
Adds mean reversion. Forecast σ̂_{t+h} → σ_∞.
arch_model(r*100, vol="GARCH", p=1, q=1).fit()
Implied vol (§26)
Solve σ: BS(σ) = market_price Brent on f(σ)=BS(σ)−mkt; σ ∈ [1e-6, 5]
Forward-looking; surface = vol by strike × expiry. Use Brent over Newton at boundaries.
Sample covariance (§27)
Σ̂ = cov(R, rowvar=False, ddof=1)
Gold standard ONLY when T >> N. T < N → singular.
Ledoit-Wolf shrinkage (§28)
Σ̂_shrink = δ·F + (1−δ)·S δ* = max(0, min(1, π/γ²))
Regularises Σ when N≈T. δ ≈ 0.1–0.4 for daily equity.
LedoitWolf().fit(R).covariance_
PSD repair (§29)
Σ = QΛQᵀ; clip λ_i to ε Σ_psd = Q·diag(max(λ,ε))·Qᵀ rescale diagonal to preserve variances
Standard fix when cov is "almost-PSD" (numerical noise, missing data, bumps).

◆ VII — Capital Frameworks

Basel SA-CCR (§34)
EAD = 1.4 · (RC + PFE) RC = max(V−C, 0) PFE = mult · Σ AddOn_AC mult = min(1, 0.05 + 0.95·exp((V−C)/(2·Σ AddOn)))
Replaces CEM. Counterparty exposure for OTC + SFTs.
Basel IMA legacy (§35)
MRC = max(VaR_t, k·avg₆₀VaR) + max(sVaR_t, k·avg₆₀sVaR) k ∈ [3.0, 4.0] (traffic light)
Phased out by FRTB Jan 2025. Still used in some emerging-market regulators.
FRTB SA (SBM) (§36)
Delta: √( Σ WS_b² + Σ_{b≠c} ρ_bc·WS_b·WS_c ) Vega: same on vega sens Curvature: max(CVR_up, CVR_down)² aggregated + DRC + RRAO
Sensitivity-tree: 5 risk classes × dozens of buckets × per-bucket aggregation.
FRTB IMA (§37)
IMCC = 0.5·ES_full_stressed + 0.5·ES_reduced ES at α=0.975 with LH scaling IMA = IMCC + SES (NMRF) + DRC
ES(97.5) ≈ VaR(99) under Normal. Desk-level PLA + backtest required.
Operational cost: most banks dropped IMA for many desks; SA fallback.
IRC / DRC (default risk) (§38)
IRC: MC on Vasicek ASRF, 99.9%/1y DRC: JTD_i = LGD·notional − MtM DRC = Σ_buckets max(Σlongs, |Σshorts|)
Default = jump, missed by VaR. Concentrated single-names get hammered.

◆ Top 6 traps (6-month memory)

  • Units must match between s and Σ (bp vs decimal vs %). Always check σ_pnl in book units.
  • DV01 sign convention — pick ∂V/∂r or loss-given-up; label EVERY report.
  • 250-day window at α=0.995 → VaR = 2nd worst day → enormous sampling error.
  • Gamma uses F (forward), not S (spot) in Black-76 denominator.
  • Always set MC seed — risk numbers must be reproducible day-to-day.
  • VaR not subadditive → can punish diversification on heavy-tail / discrete-loss books. Use ES for capital.
