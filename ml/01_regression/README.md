# 01 — Regression

Linear, regularised, and GLM-family regression — simplest entry point into
supervised ML.

## Files

- [`regression.ipynb`](regression.ipynb) — narrative walkthrough.
- [`regression_cheatsheet.md`](regression_cheatsheet.md) — code-first reference.

## What's covered

- OLS, ridge, lasso, elastic net.
- Polynomial features + regularisation.
- Diagnostics: residual plots, leverage, Cook's distance.
- Cross-validation + hyperparameter selection.
- When to reach for GLM (Poisson, gamma) instead of OLS.

## Prerequisites

- `fundamentals/mathematics.ipynb` for linear algebra refresher.
- Basic numpy + pandas comfort.
