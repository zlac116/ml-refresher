# 02 — Classification

Discriminative models for categorical targets — binary and multiclass.

## Files

- [`classification.ipynb`](classification.ipynb) — narrative walkthrough.
- [`classification_cheatsheet.md`](classification_cheatsheet.md) — code-first reference.

## What's covered

- Logistic regression (binary + multinomial).
- Tree-based: decision trees, random forests, gradient boosting (XGBoost / LightGBM).
- Naive Bayes (text + numeric).
- Evaluation: accuracy vs precision/recall/F1, ROC + PR curves, class imbalance handling, threshold tuning.
- Calibration (Platt, isotonic).
